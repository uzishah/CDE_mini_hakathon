# Banggood Data Analysis Mini Hackathon Project

## Project Overview
This project focuses on **scraping, cleaning, analyzing, and visualizing product data from Banggood**. The main goal is to provide insights about product categories, ratings, reviews, pricing, and best value products using **Python, Pandas, Matplotlib, and Streamlit**.

---

## Features

1. **Data Collection & Cleaning**
   - CSV data from Banggood containing multiple categories of products.
   - Cleaned price, rating, and review columns for accurate analysis.
   - Dropped rows with missing or invalid data.

2. **Data Analysis**
   - Average rating per category.
   - Best value products (rating / price ratio) per category.
   - Product count per category.
   - Top 10 most reviewed products.

3. **Visualizations**
   - Interactive bar charts for ratings, product counts, and best value per category.
   - Horizontal bar chart for top reviewed products.
   - Visualizations created using `matplotlib` integrated with Streamlit.

4. **Deployment**
   - Interactive web app built with **Streamlit**.
   - GitHub repository connected to Streamlit Cloud for live deployment.
   - Users can view insights and charts directly on the web.
